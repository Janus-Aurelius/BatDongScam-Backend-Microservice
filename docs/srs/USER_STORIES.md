# 2.6. User Story

* As a Guest, I want to create an account so I can access platform features.
* As a User, I want to log in securely using credentials to receive a JWT token.
* As an Owner, I want to list my property with details and images.
* As a Customer, I want to search for properties by location, price, and type.
* As a User, I want to see full details and media of a specific property.
* As a Customer, I want to book a viewing session for a property.
* As an Owner/Agent, I want to manage viewing requests.
* As an Agent, I want to generate a rental contract draft from a template.
* As a Customer, I want to sign a deposit contract to reserve a property.
* As a User, I want to pay for contracts or services using PayOS/Payway.
* As a System, I want to receive payment confirmations from external gateways.
* As a User, I want to report suspicious listings or behavior.
* As an Admin, I want to review reports and take disciplinary actions.
* As a User, I want to receive real-time alerts for updates (appointments, payments).
* As a User, I want to receive important documents and receipts via email.
* As an Admin, I want to see monthly revenue and transaction statistics.
* As a User, I want to manage my personal information and avatar.
* As an Owner, I want to pay to rank my property higher in search results.
* As an Owner, I want to de-list my property once sold or rented.
* As an Admin, I want to assign a Sales Agent to manage an Owner's property.
* As a User, I want to select locations from a standardized hierarchical list.
* As a Customer, I want to save properties to my favorites for later review.
* As a User, I want to stay logged in without re-entering credentials.
* As a Customer, I want to chat in real-time with an Agent about a property.
* As an Owner, I want to see an estimated market value for my property based on historical data.
* As a Customer, I want to view a property's layout in 3D AR using my smartphone.
* As a User, I want to search for properties by drawing a custom area on a map.
* As a Buyer, I want my funds to be held in escrow until the property title is verified.
* As a User, I want to invite friends using a referral link to earn platform rewards.
* As a Customer, I want to rate and review Sales Agents after a transaction.
* As an Owner, I want to see how many people viewed or favorited my listings.
* As a User, I want my identity and land documents to be scanned automatically for faster verification.
* As a Sales Agent, I want to subscribe to premium tiers to list more properties.
* As a Global User, I want to switch between Vietnamese and English languages.
* As an Admin, I want to see a detailed audit trail of all sensitive system actions.
